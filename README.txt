Code used for various analyses within "Deep Metabolic Phenotyping of Newborn Cord Blood Reveals Maternal-Fetal Interactions and Disease Risk"

BPD_caffeine.py is the code used to analyze BPD using caffeine metabolites
drug_predictive_models.py is the code used to generate ML models to predict maternal medication use from metabolites
GA_model.py is the code to generate ML models to predict GA from metabolites
lung_surfactant.py is the code to generate ML models to predict newborn surfactant use
pathway_enrichment_analysis.R is the code for enrichment analysis
